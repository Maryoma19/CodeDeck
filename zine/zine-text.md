## Title

CodeDeck

## Short Description

A tiny C++ desk gadget for coding focus.

## Main Text

CodeDeck is a physical focus controller for programmers. It uses buttons, LEDs, and a small display to manage coding sessions without opening another app. It runs on embedded C++ and is designed to sit on my desk while I work.

## Feature Callouts

- Focus and break timer modes
- Physical buttons instead of app distractions
- LED status indicators
- Small OLED display
- C++ firmware
- Custom desk case

## Personal Line

I made CodeDeck because I code a lot and wanted a real hardware tool connected to my daily programming life.
