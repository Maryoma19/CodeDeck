## Short Description

CodeDeck is a small C++ desk gadget for coding focus. It uses buttons, LEDs, and an OLED display to run focus timers, break timers, and quick status modes without opening another app.

## Longer Description

CodeDeck is a programmable desk controller for coding sessions. It is built around an Arduino-style microcontroller and uses embedded C++ firmware, physical buttons, LED indicators, and a small OLED display. The goal is to make a real desk device that helps me stay focused while coding instead of relying on another app or website.

I made this project because I code a lot, especially in C++, and I wanted my hardware project to connect to my daily programming life. I wanted it to feel like a small product: a case, controls, screen, wiring, firmware, and a clear purpose.

## What I Designed

I designed the CodeDeck project concept, repository, README, BOM, C++ firmware starter, Wokwi wiring diagram, CAD case layout, and Fallout zine page. The CAD design includes the main enclosure layout, screen opening, button holes, LED holes, screw posts, and internal electronics placement.

The repository includes the README, BOM CSV, Arduino/C++ firmware source, wiring diagram, CAD screenshots, and Fallout zine page.
