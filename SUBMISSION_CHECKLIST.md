# CodeDeck Fallout Submission Checklist

Use this before submitting the design. Do not submit until every required item is actually present in the GitHub repository.

## README

- [ ] Explains what CodeDeck is
- [ ] Explains what CodeDeck does
- [ ] Explains why I made it
- [ ] Explains how to use it
- [ ] Includes images of the digital design
- [ ] Links to the BOM
- [ ] Links to firmware files
- [ ] Links to CAD/wiring files

## Design Files

- [ ] Complete CAD assembly exists
- [ ] CAD includes the case
- [ ] CAD includes the buttons
- [ ] CAD includes the display
- [ ] CAD includes LED holes or light pipes
- [ ] CAD includes a real way to mount parts, such as screws, clips, or brackets
- [ ] CAD has no floating parts
- [ ] CAD source file or Onshape link is included
- [ ] STEP export is included

## Electronics

- [ ] Wiring diagram is included if using wires
- [ ] PCB source files are included if using a PCB
- [ ] Microcontroller pins match the firmware
- [ ] OLED display wiring is shown
- [ ] Button wiring is shown
- [ ] LED resistor wiring is shown

## Firmware

- [ ] Firmware source code is included
- [ ] Firmware has focus mode
- [ ] Firmware has break mode
- [ ] Firmware reads buttons
- [ ] Firmware controls LEDs
- [ ] Firmware updates the display
- [ ] Required libraries are mentioned

## BOM

- [ ] BOM is a CSV file
- [ ] Every part has a quantity
- [ ] Every part has a purpose
- [ ] Every part has a link when possible
- [ ] Parts match the actual design

## Zine Page

- [ ] A5 dimensions: 5.83 x 8.27 inches
- [ ] Uses the official template
- [ ] Describes what CodeDeck is
- [ ] Has a central graphic of the whole design
- [ ] Includes QR code to the GitHub repository
- [ ] Includes name/alias
- [ ] Includes age
- [ ] Includes country/state
- [ ] Added to the README

## Sanity Check

- [ ] Someone else looked over the design
- [ ] The project is original
- [ ] No AI-generated graphics or design files are submitted
- [ ] No copied tutorial design is presented as my own
- [ ] Repo folders and filenames are clear
