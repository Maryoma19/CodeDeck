# CAD Files

Put the CodeDeck CAD files here.

Required before submitting:

- CAD source file, such as `.f3d`, `.FCStd`, or an Onshape link
- Exported `.step` file
- Screenshots of the fully assembled model

Recommended design:

- Rectangular desk enclosure
- Top cutout for a 0.96 inch OLED display
- Three button holes labeled Mode, Start/Pause, and Reset
- Three LED holes labeled Focus, Break, and Done
- USB cable opening on the side
- Screw posts or clips inside to hold the board and close the case

Important: make this CAD yourself. Fallout says design files and graphics should not be AI-generated.
